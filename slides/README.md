# Slides

정적 HTML/CSS/JS 기반 발표 deck입니다.

## Files

- `index.html`
  - 메인 deck 엔트리
- `styles.css`
  - 시각 시스템과 레이아웃
- `app.js`
  - 키보드/버튼 네비게이션과 progress UI

## Open

가장 간단한 방법:

```bash
open slides/index.html
```

로컬 서버로 열기:

```bash
python3 -m http.server 8000
```

그 다음 브라우저에서 `http://localhost:8000/slides/`에 접속합니다.

## Notes

- 현재 차트와 contour는 발표 구조를 위한 placeholder visual입니다.
- 실제 데모 결과가 나오면 Slide 8~10의 시각 요소를 교체하면 됩니다.
