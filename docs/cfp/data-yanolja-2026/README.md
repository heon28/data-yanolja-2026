# Data Yanolja 2026 CFP Docs

이 폴더는 데이터야놀자 2026 발표 제안과 발표 준비를 위한 공개 문서를 담습니다.

## Files

- `../../../slides/index.html`
  - 실제 발표용 HTML slide deck
- `cfp-submission.md`
  - 제출 폼에 바로 옮겨 적을 수 있는 발표 제안 문안
- `slide-outline.md`
  - 발표 자료의 전체 흐름과 슬라이드별 핵심 메시지
- `slides-content.md`
  - 슬라이드별 제목, 화면 문구, 발표자 설명 초안
- `slides-screen-copy.md`
  - 실제 슬라이드 화면에 올릴 수 있도록 압축한 문구 모음
- `frontend-slides-brief.md`
  - HTML 슬라이드 제작 시 필요한 레이아웃, 톤, 디자인 토큰, 슬라이드별 구성 브리프
- `demo-spec.md`
  - 합성 CRM 시뮬레이터와 데모 설계 문서

## Public Scope

- 이 폴더의 문서는 GitHub에 공개 가능한 제출용/공유용 문서입니다.
- 내부 브레인스토밍, 설계 메모, 실행 계획은 `docs/superpowers/`에서 별도로 관리합니다.
